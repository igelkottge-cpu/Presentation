# Урок 1 — плоская версия для переноса в Figma

Все 10 слайдов идут вертикально, каждый ровно 1280×720. Декор и портрет «запечены» в фон каждого слайда, поэтому в Figma приходит минимум слоев. Тексты и карточки остаются отдельными редактируемыми элементами.

## Деплой на GitHub Pages
1. Загрузите содержимое этой папки в КОРЕНЬ репозитория (чтобы `index.html` лежал в корне).
2. Settings → Pages → Source: ветка `main`, папка `/ (root)`.
3. Через минуту страница откроется по адресу `https://<логин>.github.io/<репозиторий>/`.

## Перенос в Figma
1. Откройте задеплоенную страницу.
2. Дайте шрифту Gilroy прогрузиться (пара секунд).
3. Запустите плагин html.to.figma один раз — заберет сразу все 10 слайдов.

## Структура
- `index.html` — страница со слайдами
- `backgrounds/` — запеченные фоны bg-01…bg-10.png
- `fonts/` — шрифт Gilroy
- `.nojekyll` — отключает обработку Jekyll
