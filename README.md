# Урок 1. Почему Макс — лучший выбор для репетитора

Презентация-скринкаст (10 слайдов).

## Деплой на GitHub Pages
1. Загрузите содержимое этой папки в репозиторий (файлы должны лежать в корне ветки или в папке `/docs`).
2. Settings → Pages → Source: ветка `main`, папка `/ (root)` (или `/docs`, если положили туда).
3. Через минуту страница откроется по адресу вида `https://<логин>.github.io/<репозиторий>/`.

## Структура
- `index.html` — презентация (открывается автоматически)
- `deck-stage.js` — движок слайдов
- `fonts/` — шрифт Gilroy (6 начертаний)
- `assets/` — изображения
- `.nojekyll` — отключает обработку Jekyll

## Навигация
Стрелки ← / → или клик по краям. Печать в PDF — Cmd/Ctrl + P.
